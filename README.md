### SwiftJSONGenerator
将JSON格式化输出为模型的属性　　　　

####怎么安装：

方式1：下载SwiftJSONGenerator.zip

方式2：解压后放到应用程序中`~/Applications` 打开


JSON格式
~~~
{
    "name": "王五",
    "gender": "man",
    "age": 15,
    "height": "140cm"
}
~~~

~~~
{
    "name": "王五",
    "gender": "man",
    "age": 15,
    "height": "140cm",
    "addr": {
        "province": "fujian",
        "city": "quanzhou",
        "code": "300000"
    },
    "hobby": [
        {
            "name": "billiards",
            "code": "1"
        },
        {
            "name": "computerGame",
            "code": "2"
        }
    ]
}
~~~
